# YT_playlist_watch_hours
Calculate watch hours of youtube playlist

If you are on a computer, you can use a line of code to calculate it directly on the page.
(Note: If the playlist is very long, scroll all the way to the bottom of the page first so all the videos load).
Open your playlist link in a web browser (Chrome, Edge, Firefox, Safari).
Right-click anywhere on the empty space of the page and select "Inspect".
At the top of the menu that opens, click the "Console" tab.


Once you have the Console tab open, here is exactly where and how to paste the code:
1. Look for the prompt symbol (>)
Scroll to the very bottom of the Console window. You are looking for a little blue (or grey) right-facing arrow > with a blinking cursor next to it. This is the text input line.
2. Click next to the arrow
Click your mouse right next to that > symbol to make sure your cursor is active there.
3. Paste the code
Press Ctrl + V (Windows/Linux) or Cmd + V (Mac) to paste the code block.
⚠️ IMPORTANT: Chrome's "Allow Pasting" Security Warning
If you are using Google Chrome or Microsoft Edge, you might get a warning message the first time you try to paste something here. The browser does this to protect people from scammers.
If it blocks you from pasting, look at the message it gives you.
It will ask you to manually type the words allow pasting into the console and hit Enter.
Once you type allow pasting and hit Enter, you can press Ctrl + V to paste the code again.
4. Hit Enter to run
Once you see the block of code sitting next to the > symbol, just press the Enter key on your keyboard.
